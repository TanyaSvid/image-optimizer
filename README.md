Умный Оптимизатор Изображений (Smart Image Optimizer)

Минималистичное и быстрое веб-приложение для оптимизации изображений (JPEG и PNG), работающее полностью на стороне клиента. Идеально подходит для массовой обработки графики, например, для интернет-магазинов, где важно сохранять структуру папок по артикулам.

✨ Ключевые особенности

Сохранение структуры папок: Поддерживает массовую загрузку папок через Drag & Drop. Скачанный ZIP-архив полностью повторяет изначальную архитектуру директорий (файлы из разных папок не перемешаются).

Локальная обработка (Privacy First): Изображения не отправляются на сторонние серверы. Вся магия сжатия происходит прямо в вашем браузере с использованием HTML Canvas API.

Поддержка тяжелых файлов: Обрабатывает файлы весом до 10 МБ.

Умное сжатие: Автоматически подбирает алгоритм. Если сжатый PNG получается больше оригинала (что бывает с мелкими файлами), скрипт сохраняет исходник.

Удобный экспорт: Выгружает все обработанные изображения в один клик в формате .zip.

Без зависимостей (Zero-backend): Состоит из одного файла index.html. Никакого Node.js, баз данных или сложной настройки.

🛠 Технологии

HTML5 / JavaScript (ES6)

Tailwind CSS — для стилизации интерфейса (подключается через CDN).

JSZip — для генерации итогового архива на лету.

File System API — для рекурсивного чтения загружаемых папок.

🚀 Как использовать

Вариант 1: Локально на компьютере

Скачайте или клонируйте этот репозиторий.

Откройте файл index.html в любом современном браузере.

Перетащите нужные папки с картинками в зону загрузки.

Вариант 2: Онлайн (GitHub Pages)

Проект можно бесплатно захостить прямо на GitHub:

Зайдите в настройки репозитория (Settings).

Перейдите в раздел Pages.

В поле Source выберите ветку main (или master) и нажмите Save.

Через пару минут оптимизатор будет доступен по ссылке https://<ваш-юзернейм>.github.io/<имя-репозитория>.
